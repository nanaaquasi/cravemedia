# Frontend Design Examples

## Example 1: Brutalist Landing Hero

**Context**: SaaS product for developers. Tone: raw, utilitarian, confident.

**Choices**:
- Font: Bebas Neue (display) + IBM Plex Mono (body)
- Color: Near-black (#0a0a0a), white text, single accent (#ff3e00)
- Layout: Asymmetric grid, large type breaking into negative space
- Motion: Staggered fade-in on load (0.1s increments)

## Example 2: Editorial Magazine Section

**Context**: Long-form article layout. Tone: refined, readable, editorial.

**Choices**:
- Font: Playfair Display (headings) + Source Serif Pro (body)
- Color: Cream (#faf8f5), dark charcoal (#1a1a1a), subtle gold accent
- Layout: Narrow column (65ch), generous margins, drop caps
- Motion: Minimal—subtle fade on scroll into view

## Example 3: Retro-Futuristic Dashboard

**Context**: Analytics dashboard. Tone: sci-fi, data-dense, immersive.

**Choices**:
- Font: Orbitron (headings) + Exo 2 (body)
- Color: Dark (#0d1117), neon cyan (#00fff5), grid lines
- Layout: Dense grid, overlapping cards, diagonal accents
- Motion: Glow pulses, staggered card entrances, hover lift

---

Use these as inspiration. Each design should be unique to its context—never copy; interpret and adapt.
